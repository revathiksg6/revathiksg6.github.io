# Personal Portfolio Website

## Objective
Host a simple static website on GitHub Pages for free, as part of the Elevate Labs Web Development Internship (Task 5).

## Tools Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages

## What I Did
- Created a responsive personal portfolio website with sections for About, Skills, Projects, and Contact.
- Designed a black-themed UI with a sticky navbar, gradient-free dark layout, and card-based skills/projects sections.
- Added smooth scrolling and dynamic navbar behavior using JavaScript.
- Pushed the code to a GitHub repository named `revathiksg6.github.io`.
- Enabled GitHub Pages in repository settings to deploy the site live.
